# app-centralizacao-senai

Aplicativo desenvolvido para aumentar a eficiência nas atividades do curso de Desenhista Técnico no SENAI, realizando cálculos automáticos de centralização de peças na folha.

## Funcionalidades
- Calcula centralização automaticamente
- Interface simples e intuitiva
- Geração de resultados precisos
  

## Como usar
1. Abra o aplicativo senaio.exe
2. Insira os dados do peça a ser desenhada e medidas da folha (sem contar a margem)
3. Clique em "Calcular" e visualize o resultado

## Tecnologias utilizadas
- Linguagem: GDScript
- Framework: Godot Engine
